Javascript Editor supports Mobello class system

Content Assist

The content assist (Ctrl+Space) in javascript editor now supports Mobello class system. You can choose functions and variables (local, global) in class or class itself. It also support many useful templates for Mobello.
If '$require' statement is required, then it will be generated automatically.

  • Usage: Input text and press Ctrl + Space

Outline View and quick hierarchy

The outline view and quick hierarchy (Ctrl+O) now supports Mobello class system. You can view the structure of a class quickly.

  • Quick hierarchy usage: Press Ctrl + O in javascript editor

 

Syntax Highlighting

Mobello class system keywords like $class, $require, define, extend, mixin will be highlighted.

Type inferrence based on JSDoc

If a return type and parameter types is specified in JSDoc, content assist processor will infer that types based on JSDoc information.

Open Declaration

Open declaration (F3) now support Mobello class system.,

  • Usage 1: Select class name and press F3 in javascript editor.
  • Usage 2: Select class name and open context menu. Then select Open Declaration.

Type Hierarchy

Type hierarchy (F4) view now supports Mobello class system.

  • Usage 1: Select class name and press F4 in javascript editor.
  • Usage 2: Select class name and open context menu. Then select Open Type Hierarchy.

Call Hierarchy

Call hierarchy (Ctrl+Alt+H) now supports functions in Mobello class system.

  • Usage: Select method and press Ctrl+Alt+H

Template

Content assist now supports templates for creation Mobello components and controllers.

Integration with Appspresso

Appspresso is a mobile app framework which wraps source code developed with web technology by each mobile platform runtime and builds them into native mobile apps. Currently, developers can build Apple iOS apps, Google Android apps.

If you want to build hybrid mobile app with Appspresso, you have to install Appspresso to our Mobello studio.  See this installation instruction to know how to install Appsrpesso to Mobello Studio.

For more information, see Build Hybrid App with Mobello + Appspresso

New Mobello with Appspresso Project Wizard

You can create hybrid cross-platform application which able to run in both iOS and Android platform by using new project wizard. New project's structure is same as Appspresso application project, so you can utilize Appspresso's features for building mobile hybrid app.
