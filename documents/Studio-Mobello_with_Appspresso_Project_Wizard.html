Prerequisite

Mobello with Appspresso Project Wizard 를 사용하기 위해서는 Appspresso 1.1.2  이상이 설치되어야 합니다. Appspresso 설치하는 방법은 다음을 참조하시기 바랍니다.

  • Install Appspresso 1.1.2 to existing Mobello Studio 1.1

Mobello with Appspresso Project Wizard

Mobello with Appspresso Project Wizard 는 하이브리드 앱빌더인 Appspresso 를 사용해서 iOS 또는 Android 에서 실행되는 mobile app 을 만들기 위한 wizard 입니다.

Project name
Project의 이름을 지정합니다. Project 이름은 application 의 이름으로 사용되며 영문자, 숫자 와 underscore(_) 만 허용됩니다.

Title
사용자에게 노출될 Application 의 제목을 지정합니다.

Version
Application 의 버전을 지정합니다.

Vendor
Application 의 제작자를 지정합니다.

Theme
Application 의 기본 Theme 를 선택합니다. 기본적으로 default theme 와 iOS theme 가 제공됩니다.

Working sets
생성된 Project를 working set 에 추가합니다.

Project 생성에 필요한 항목을 입력하고 Finish 버튼을 클릭하면 기본 scene based application 이 생성됩니다. Next 버튼을 클릭하면 application template page 로 이동합니다.

Mobello Application Template Page

Application template page 는 생성될 application 에 적용될 template 을 선택할 수 있는 wizard page 입니다. template 은 application 유형에 따라 scene based, navigation based, tab bar 의 3가지가 제공됩니다.

Scene based template

단일한 scene(화면) 만 존재하는 가장 간단한 application 유형입니다.

Navigation based template

상단 navigation bar 를 통해 여러개의 scene 들간의 이동이 자유로운 application 유형입니다.

Tab bar template

하단 tab bar 를 통해 여러개의 scene 들간의 이동을 지원하는 application 유형입니다.
