웹 디자인에서 CSS 에 대한 지식은 절대적이지만 QuickStyler 를 활용하여 초보 개발자도 쉽게 CSS 를 사용하여 UI 를 디자인할 수 있습니다. QuickStyler 는 Text Section, Layout Section, Position and Size Section, Color Section, Border Section 등으로 구성되어 있으며 style 속성을 쉽게 편집할 수 있도록 도와줍니다.
 

Text

 
Text Section 에서는 폰트 종류, 폰트 크기, bold, italic 같은 텍스트 관련 style 을 편집할 수 있습니다.
 

  1. Scene 에 Button 컴포넌트를 배치한 후 Text section 에서 font 와 font-size, align 등을 변경해 보도록 합니다.
     

Color

 
Color Section 에서는 배경, 글자색, border 색상등을 변경할 수 있습니다.
배경의 경우에는 단일 색상, gradient, 이미지 중에서 선택할 수 있습니다. 
gradient 를 선택한 경우 하단의 gradient bar 에서 색상 위치를 정의하는 stop 을 추가/삭제/이동하고 상단의 color picker 에서 stop 의 색상을 선택할 수 있습니다.
 

  1. Button 컴포넌트의 경우 배경이 기본적으로 gradient 로 설정되어 있는데 이를 변경해 보도록 하겠습니다. 먼저 gradient bar 중간에 위치하는 stop 을 삭제하기 위해서 stop 을 마우스로 선택하고 아래쪽으로 드래그합니다. 중앙의 stop 2개를 이런식으로 제거할 수 있습니다.
     

     
  2. 가장 오른쪽 stop 을 선택한 후 color 값을 다음과 같이 파란색으로 변경합니다.
     


     
  3. 글자 색생을 변경하기 위해서는 Foreground Color 를 선택합니다.
     

     
  4. Swatches 에서 배경에 어울리는 적당한 색상을 선택해 봅니다.
     

     
  5. 만일 기본 swatch 에 원하는 색상이 없다면 swatch 종류를 변경하거나 solid 에서 색상을 선택합니다.
     

Border

 
Border Section 에서 border 의 width, style을 지정하거나 라운딩 처리를 수행할 수 있습니다.
 

  1. Border section 에서 width 값은 5 로, 라운딩 처리에 해당하는 radius 값은 16 으로 변경해 보고 결과를 designer 에서 확인해 봅니다.
     

     
  2. 만일 border 의 style 을 변경하려면 버튼을 클릭하고 4 방향의 border 속성을 각각 다르게 지정하려면 버튼을 사용합니다.
