Mobello Application 을 debug 모드로 실행하면 breakpoint 를 설정해서 특정 위치에서 멈추도록 하고 그 위치에서의 변수값을 볼 수 있습니다. Debug 모드로 실행 중인 앱이 breakpoint 에 걸려 중단되면 debug perspective 로 전환할지 물어봅니다. 이때 debug perpsective 로 전환하면 다음과 같은 debug perspective 가 나타납니다.
 

 
1: Debug View
현재 중단된 javascript thread 의 stack frame 를 보여주고 앱과 web server 의 실행을 제어할 수 있습니다.
 
2: Variables View
Debug view 에서 선택된 stack frame 에 대한 variable 정보를 보여줍니다.
 
3: Breakpoints View
Workspace 에서 설정한 breakpoint 들을 모두 보여줍니다.
 
4: Console View
앱이 실행되는 동안 출력하는 로그와 에러 메시지가 출력됩니다.
