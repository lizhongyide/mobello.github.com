Mobello Dashboard Project

 
Mobello Dashboard Project 는 여러개의 application 들로 이루어지며 사용자가 application 을 선택해서 실행할 수 있는 다음과 같은 dashboard 화면을 제공합니다.
 

 
Mobello Dashboard Project 의 기본 구조는 다음과 같습니다.
 

 
Dashboard project 에 대한 설정 파일인 config.json 과 app 실행시 bootstrapper 역할을 하는 launcher.html 은 반드시 project root 에 존재해야 합니다. Project 를 구성하는 app 들은 apps 폴더 안에 존재하면 app 폴더 이름은 application 이름과 같아야 합니다. app 폴더안의 내용은 Application Project 의 구조와 같습니다. 여러 application 들이 함께 사용하는 파일들은 shared 폴더에 두면 공유해서 사용할 수 있습니다.

브라우저로 실행하기

 
Web app 으로 publish 된 dashboard project 를 크롬 브라우저에서 실행하려면 launcher.html 파일을 브라우저에서 열면 됩니다.
Dashboard project 가 아닌 application 을 브라우저에서 바로 실행하기 위해서는 주소창에 다음과 같이 ?app=app 이름 형식으로 parameter 를 붙여주어야 합니다.
