현재 일시정지된 javascript thread 의 stack frame 정보를 보여주고 앱과 web server 의 실행 상태를 확인하고 제어할 수 있습니다.
 

 
1: Thread stack frame
현재 일시 정지된 javascript thread stack frame 들을 트리 형태로 보여줍니다.
 
2: Embedded Web Server
앱이 동작하고 있는 web server 의 실행 상태를 보여줍니다. 선택된 상태에서 Terminate 버튼으로 web server 를 중지시킬 수 있습니다.
 
3: Remove All Terminated Launches
종료된 앱들을 debug view 에서 모두 제거합니다.
 
4: Resume
일시 중지중인 thread 를 다시 재개합니다.
 
5: Suspend
선택된 thread 를 일시중지 시킵니다.
 
6: Termiate
선택된 process 나 web server 를 종료시킵니다.
 
7: Disconnect
선택된 process 와의 debug 연결을 끊습니다.
 
8: Step Into
현재 중지된 라인에 존재하는 function 내부로 진행합니다.
 
9: Step Over
현재 중지된 라인의 다음 라인으로 진행합니다.
 
10: Step Out
현재 중지된 라인이 속한 function 을 빠져 나옵니다.
 
11: Drop To Frame
Stack frame 을 선택하고 이 버튼을 클릭하면 이 frame 에서 부터 다시 돌아가서 진행할 수 있습니다.
 
12: Step Filter
Step filter 기능을 끄거나 켭니다.
