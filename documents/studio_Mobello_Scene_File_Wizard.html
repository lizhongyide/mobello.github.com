Mobello scene file wizard 를 통해서 application 에 scene file 을 추가할 수 있습니다.

Enter or select the parent folder
Scene file 이 생성될 folder 의 경로를 입력하거나 project 에서 선택할 수 있습니다.

File name
생성될 scene file 의 이름을 입력합니다. 확장자를 입력하지 않아도 자동으로 scene 이 확장자로 붙습니다.

Scene file wizard 를 사용해서 scene file 을 생성하면 scene 에 대한 controller 역할을 하는 js 파일도 자동으로 생성되게 됩니다.
