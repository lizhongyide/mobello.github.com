Flow layout 은 웹의 가장 기본적인 layout 으로 주로 텍스트 위주의 문서에 적합 합니다. Flow layout 에서는 flow type 이 inline 이나 inline-block 인 child 컴포넌트는 수평으로 배치되며 flow type 이 block 인 child 컴포넌트는 하나의 컴포넌트가 라인 전체를 차지하게 됩니다. 다음은 flow layout 안에서 flow type 이 inline-bock 인 button 들 사이에 block 인 button 을 배치한 예제입니다.
 

Flow type 전환하기

  1. 비어있는 Scene 에 Panel 을 배치합니다. Panel 의 layout 값이 flow layout 인 것을 확인합니다.
  2. 배치된 Panel 에 Button 컴포넌트를 여러개 배치합니다.
  3. Button 컴포넌트중 하나를 선택하고 QuickStyler > Position and Size 에서 flow type 을 다음 값들로 변경해 보겠습니다.
    block
    컴포넌트가 수평 라인 전체를 차지합니다.
    inline
    화면의 수평 라인을 따라 배치되며 공간이 없으면 다음 라인에 배치됩니다. 일반적인 텍스트의 flow type 이 inline 입니다. Inline 컴포넌트에는 width 와 height 가 적용되지 않습니다. 따라서 텍스트 이외의 컴포넌트를 inline 으로 지정하면 제대로 렌더링 되지 않을 수 있습니다.
    inline-block
    inline 과 같이 수평 라인을 따라 배치되지만 width 와 height 가 적용됩니다.

Designer 에서 width/height/margin 변경하기


Flow type 이 block 이나 inline-block 인 컴포넌트는 width/height 를 마음대로 변경할 수 있습니다.

  1. 컴포넌트를 선택하면 나타나는 8 개의 흰색 사각형 handle 을 사용해서 크기를 조정해 보겠습니다.



    만일 width/height 비율을 유지한채 크기를 변경하려면 키보드의 shift 키를 누른 상태에서 handle 을 마우스로 드래그를 하면 됩니다.
  2. Margin 은 컴포넌트간의 간격을 지정하는 속성입니다. Designer 에서 margin 을 변경하기 위해서는 컴포넌트를 선택한 후 노란색 점선을 드래그하면 됩니다.

    top margin 을 늘이기 위해서는 top 영역의 노란선을 아래쪽으로 마우스 드래그 합니다.



    드래그가 완료되면 다음과 같이 margin 이 적용된 것을 볼 수 있습니다.


    left/right/bottom margin 도 같은 방법으로 조정할 수 있습니다.

 

QuickStyler 에서 width/height/margin/padding 조정하기


Designer 에서 뿐만 아니라 QuickStyler 의 Position and Size section 에서도 width/height/margin/padding 값을 변경할 수 있습니다.



width/height/margin 은 px 값이나 % 값을 가질 수 있으며 auto 값을 가질 수 도 있습니다. px ↔ % ↔ auto 간의 전환은 각 항목의 오른쪽 버튼을 클릭하면 됩니다. 
border 안의 여백을 나타내는 padding 은 다른 값들과 달리 auto 값은 가질 수 없습니다. width/height/margin/padding/border 속성의 의미는 다음 그림을 통해 쉽게 이해 가능합니다.

  1. 먼저 선택된 컴포넌트의 width 를 100px 로 변경해보겠습니다. Width 항목의 오른쪽 버튼을 클릭해서 px 로 변경한 후 직접 입력하거나 위/아래 방향키로 값을 조절할 수 있습니다.
  2. 여기서 버튼을 한번더 클릭하면 designer 에서의 크기는 그대로 유지되면서 % 값으로 변경되는 것을 볼 수 있습니다.


    height 도 마찬가지 방법으로 쉽게 값을 변경할 수 있습니다.
  3. margin 과 padding 은 top/right/bottom/left 의 값을 각각 변경할 수 있습니다. top/right/bottom/left 항목을 펼치기 위해서는 Margin 라벨 옆 삼각형을 클릭합니다.
  4. margin 의 top 값을 선택해서 변경하면 다음과 같이 margin 항목에 반영되는 것을 볼 수 있습니다.
  5. 여기서 Margin 항목 오른쪽 Reset 버튼을 클릭하면 모든 margin 값이 0px 로 초기화 됩니다.
