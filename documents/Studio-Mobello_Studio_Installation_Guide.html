Mobello Studio에는 Mobello Framework이 내장되어 있어 별도로 설치할 필요가 없습니다.

Mobello Studio 요청하기

KT 에코노베이션 센터에서 요청하기

  1. 에코노베이션의 모벨로 페이지(http://econovation.co.kr/mobello/)에 접속합니다.
  2. 상단의 "Studio" 탭을 선택합니다.
  3. "Studio 다운로드" 버튼을 선택합니다.
  4. 이메일 주소를 기입한 후 약관동의 버튼을 선택합니다.
  5. 전송하기 버튼을 선택합니다.
  6. 전송완료 창이 나타납니다.

Mobello 홈페이지에서 요청하기

  1. Mobello 홈페이지(http://mobello.github.com/)에 접속합니다.
  2. "Download Mobello Studio" 버튼을 선택합니다.
  3. Mobello 사이트의 "다운로드 페이지"에서 이메일을 입력합니다.

Mobello Studio 다운로드

  1. Mobello Studio의 다운로드 링크가 입력한 이메일로 발송됩니다.
  2. 메일로 전송 된 링크 중 자신의 환경에 맞는 링크를 선택하여 다운로드 받습니다.

 

Mobello Studio 실행하기

  1. 다운로드받은 설치 파일의 압축을 해제합니다.
  2. studio.exe 파일을 실행합니다.
  3. 프로젝트들이 저장될 workspace을 지정합니다.
  4. 스튜디오가 성공적으로 설치가 완료된 경우 Mobello Studio가 실행되면서 Welcome 페이지가 나타납니다.

Welcome 페이지에서는 Studio 에서 제공하는 예제 프로젝트들을 import 할 수 있습니다. 또한 사용자 가이드문서나 동영상 가이드를 확인할 수 있으며 Open Cheat Sheet 를 클릭하면 예제를 따라해볼 수 있는 cheat sheet 를 실행할 수 있습니다.
