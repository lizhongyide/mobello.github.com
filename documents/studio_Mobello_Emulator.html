Mobello Emulator 는 모바일 디바이스 필요 없이 PC 에 설치된 크롬 브라우저를 통해서 작성한 application 을 테스트하고 디버깅할 수 있도록 해줍니다.
 

 
1
현재 emulator 에서 실행중인 앱을 스마트폰에서 직접 실행해 볼 수 있도록 다음과 같이 접속 URL 과 QR 코드를 제공 합니다.
 

 
2:Home
첫 실행 화면으로 돌아갑니다.
 
3:Rotate
Emulator 를 세로가 긴 portrait 화면 모드와 가로가 긴 landscape 화면 모드간에 전환합니다.
 
4:Change Device
Emulator 배경 화면을 iPhone 화면과 Android Nexus One 화면간에 전환합니다.
 
5:Zoom
Zoom In 으로 화면을 확대하거나 Zoom Out 으로 축소할 수 있습니다.
 
6:Close
에뮬레이터를 종료합니다.
 
Chrome Developer Tools
Chrome 브라우저에서 제공하는 개발자용 콘솔(Chrome Developer Tools) 를 사용해서 디버깅을 하거나 에러 메시지를 확인할 수 있습니다. 개발자용 콘솔은 키보드 F12 나 Ctrl + Shift + J 로 띄울 수 있습니다. 개발자용 콘솔에 대한 자세한 사용법은 http://code.google.com/chrome/devtools 에서 확인할 수 있습니다.
