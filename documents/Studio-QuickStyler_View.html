QuickStyler view 는 스타일 관련 속성들을 쉽게 수정할 수 있는 UI 를 제공합니다. QuickStyler 는 여러개의 section 으로 구성되어 있으며 컴포넌트에 따라 나타나는 section 이 다를 수 있습니다.
 

Text section

 

 
1: Font Family
Font를 선택합니다. 선택 항목 중 마지막 ADD... 를 선택하면 Window > Preferences > Visual Designer > Font Families page 가 뜨고 선택할 수 있는 Font 의 종류를 편집할 수 있습니다.
 
2: Font Size
Font 크기를 pixel 단위로 선택합니다.
 
3: Align Left
텍스트를 왼쪽으로 정렬합니다.
 
4: Align Center
텍스트를 중앙으로 정렬합니다.
 
5: Align Right
텍스트를 오른쪽으로 정렬합니다.
 
6: Bold
텍스트에 bold 스타일을 적용합니다.
 
7: Italic
텍스트에 italic 스타일을 적용합니다.
 
8: Underline
텍스트에 underline 스타일을 적용합니다.
 
9: Line Height
텍스트의 줄간격을 변경합니다. 숫자 또는 pixel 값을 적용할 수 있습니다.
 

Color Section

 
Color Section 에서 배경색, 텍스트색, border 색상을 변경할 수 있습니다. 변경할 색상을 선택한 후 하단의 Swatchs, Solid, Graident, Image 중 한곳에서 색상을 변경합니다.
 

 
1: Background Color
배경색을 선택합니다. Reset 버튼으로 배경을 초기화할 수 있습니다.
 
2: Foreground Colo
텍스트색상을 선택합니다. Reset 버튼으로 텍스트색을 초기화할 수 있습니다.
 
3: Border Color
Border 색을 선택합니다. Reset 버튼으로 border 색을 초기화할 수 있습니다.
 

Swatches

 
미리 정의된 swatch 에서 색상을 선택합니다. 또한 색상을 ==transparent(투명)== 으로 변경할 수 있습니다.
 
4
Swatch 의 종류를 선택합니다.
 
5
Swatch 를 추가합니다. http://www.colorzilla.com/firefox/palettes.html 에서 swatch 정의 파일을 받아서 추가할 수 있습니다.
 
6
선택된 색상을 transparent 로 변경합니다.
 

Solid

 
단일 색상을 선택할 수 있습니다.
 

 
1
이곳에서 원하는 색상을 마우스로 선택합니다.
 
2
1번에서 출력되는 색상의 기본 색상을 지정합니다.
 
3,4,5
선택된 색상의 RGB값을 보여주고 변경할 수 있습니다.
 
6
선택된 색상값을 web color 형식으로 보여주고 변경할 수 있습니다.
 
7
현재 선택된 색상을 나타냅니다.
 
8
색상값을 모니터 화면에서 선택합니다.
 

Gradient

 
Gradient 색상을 선택할 수 있습니다.
 

 
1
현재 선택된 stop 의 색상을 선택합니다.
 
2
1 의 기본 색상을 변경합니다.
 
3,4,5
현재 선택된 stop 의 색상 값을 보여주고 변경할 수 있습니다.
 
6: STOP
현재 선택된 stop 의 위치를 % 단위로 보여주고 변경할 수 있습니다.
 
7: WebColor
현재 선택된 stop 의 색상을 web color 형식으로 보여주고 변경할 수 있습니다.
 
8: Type
Gradient type 을 linear 와 radial 중 선택할 수 있습니다.
 
9: Direction
Gradient type 이 linear 인 경우 나타납니다. Gradient 방향을 Vertical(수직), Horizontal(수평), UpwardDiagonal(대각선 45도 오른쪽 위 방향), DownwardDiagonal(대각선 45도 오른쪽 아래 방향) 중 하나로 선택할 수 있습니다.
 
10
현재 선택된 stop 의 색상을 보여줍니다.
 
11
현재 적용된 gradient 를 미리보기로 보여줍니다. 이영역을 클릭하면 stop 이 추가됩니다.
 
12
검은 배경은 현재 선택된 stop 을 나타냅니다. stop 을 위쪽이나 아래쪽으로 마우스 드래그하면 stop 이 제거됩니다.
 
13
흰 배경은 현재 선택되지 않은 stop 을 나타냅니다. stop 을 위쪽이나 아래쪽으로 마우스 드래그하면 stop 이 제거됩니다.
 
14
graident stop 의 위치를 반전시킵니다.
 
15
stop 색상값을 모니터 화면에서 직접 선택합니다.
 

Image

 
배경 이미지를 지정합니다.
 

 
1: Background Image
배경 이미지 파일의 경로를 지정합니다.
 
2: Repeat
배경 이미지가 반복되는 방식을 정의합니다.
 
 

Border Section

 

1: Border
Border 의 두께를 지정합니다.
 
2: Radius
Border radius 의 크기를 지정합니다.
 
3: Border Style
점선, 실선과 같은 border style 을 지정합니다.

 
4: Border Detail
4방향(top, right, bottom, left) border 의 속성을 각각 설정할 수 있습니다.

 
Style
Border style 을 지정합니다.
 
Width
Border width 를 지정합니다.
 
Color
Border color 를 지정합니다.
 
 

Layout Section

 
Layout section 에는 선택된 컴포넌트의 layout 이 어떤 type 인지에 따라 다른 section 내용이 보여집니다. 또한 parent 컴포넌트의 layout type 에 따라 다른 position and size section 이 나타납니다.
 

Flow Layout

 
Layout type 이 flow 인 container 를 선택한 경우 다음과 같은 section 이 나타납니다.
 

 
1: Layout
선택된 container 의 layout type 으로 flow, absolute, flexbox 중에 하나를 선택할 수 있습니다.
 
2: Align Left
Child 컴포넌트들을 좌측 정렬합니다.
 
3: Align Center
Child 컴포넌트들을 중앙 정렬합니다.
 
4: Align Right
Child 컴포넌트들을 우측 정렬합니다.
 

Position and Size for flow layout

 
Flow layout container 안에 위치하는 컴포넌트의 경우 다음과 같은 position and size section 이 나타납니다.
 

 
1: Flow Type
Flow type 을 block, inline-block, inline 중 하나로 선택할 수 있습니다.
 
2: Width
선택된 컴포넌트의 width 를 지정합니다.
 
3: Height
선택된 컴포넌트의 height 를 지정합니다.
 
4: Margin
선택된 컴포넌트의 margin 을 지정합니다.
 
5/6/7/8: top/right/bottom/left
선택된 컴포넌트의 top/right/bottom/left margin 을 지정합니다.
 
9: Padding
선택된 컴포넌트의 padding 을 지정합니다.
 
10/11/12/13: top/right/bottom/left
선택된 컴포넌트의 top/right/bottom/left padding 을 지정합니다.
 
14/15
4방향의 margin/padding 을 각각 설정할 수 있는 입력 필드를 보여주거나 숨깁니다.
 
16
해당 속성 값을 px ↔ % ↔ Auto 간에 변환합니다. px ↔ % 간에 변환시에는 계산된 크기가 최대한 동일하게 변환됩니다.
 

Absolute Layout

 
Layout type 이 absolute 인 container 를 선택하면 다음과 같은 layout section 이 나타납니다.
 

 
1: Layout
선택된 container 의 layout type 을 flow, absolute, flexbox 중에 하나를 선택할 수 있습니다.
 

Position and size for absolute layout

 
Absolute layout container 안에 위치하는 컴포넌트의 경우 다음과 같은 position and size section 이 나타납니다. 
Section 좌측 십자선 모양의 UI 를 통해 화면 사이즈가 가변적일때, 고정 영역과 가변 영역을 지정할 수 있습니다. 파란색 직선이 화면 사이즈와 관계없이 고정되는 부분이고 빨간색 스프링이 크기가 가변적인 부분입니다. 마우스로 파란색 직선을 클릭하면 빨간색 스프링으로 변환되면서 연관된 인접 영역이 파란색 직선으로 변하게 됩니다.
 

 
1:Width
Width 를 고정 영역 ↔ 가변 영역간에 전환합니다. Right 값이 함께 전환됩니다.
 
2:Height
Height 를 고정 영역 ↔ 가변 영역간에 전환합니다. Bottom 값이 함께 전환됩니다.

3:Top
Top 을 고정 영역 ↔ 가변 영역간에 전환합니다. Bottom 값이 함께 전환됩니다.

4:Right
Right 를 고정 영역 ↔ 가변 영역간에 전환합니다. Left 값이 함께 전환됩니다.

5:Bottom
Bottom 을 고정 영역 ↔ 가변 영역간에 전환합니다. Top 값이 함께 전환됩니다.

6:Left
Left 를 고정 영역 ↔ 가변 영역간에 전환합니다. Right 값이 함께 전환됩니다.

7:Width
Width 를 지정합니다.
 
8:Height
Height 를 지정합니다.
 
9:Top
Top 을 지정합니다.
 
10:Right
Right 를 지정합니다.
 
11:Bottom
Bottom 을 지정합니다.
 
12:Left
Left 를 지정합니다.
 
13
해당 속성 값을 px ↔ % ↔ Auto 간에 변환합니다. px ↔ % 간에 변환시에는 계산된 값이 최대한 같게 변환됩니다.

14:Align Left
선택된 컴포넌트들을 가장 좌측에 있는 컴포넌트를 기준으로 정렬합니다.
 
15: Align Center
선택된 컴포넌트들을 계산된 중앙 위치로 정렬합니다.
 
16: Align Right
선택된 컴포넌트들을 가장 우측에 있는 컴포넌트를 기준으로 정렬합니다.
 
17: Align Top
선택된 컴포넌트들을 가장 위쪽에 있는 컴포넌트를 기준으로 정렬합니다.
 
18: Align Middle
선택된 컴포넌트들의 계산된 중앙 위치로 정렬합니다.
 
19: Align Bottom
선택된 컴포넌트들을 가장 아래족에 있는 컴포넌트를 기준으로 정렬합니다.
 
20: Make Horizontal Spacing Equal
선택된 컴포넌트들의 수평 간격을 동일하게 조정합니다.
 
21: Make Vertical Spacing Equal
선택된 컴포넌트들의 수직 간격을 동일하게 조정합니다.
 
22: Match Width
선택된 컴포넌트들의 width 를 동일하게 조정합니다.
 
23: Match Height
선택된 컴포넌트들의 height 를 동일하게 조정합니다.
 
24: Match Size
선택된 컴포넌트들의 width 및 height 를 동일하게 조정합니다.
 
25: Bring Forward
선택된 컴포넌트의 z-index 를 1 만큼 증가시킵니다.
 
26: Bring To Front
선택된 컴포넌트의 z-index 를 현재 container 의 child 중 가장 큰 값으로 증가시킵니다.
 
27: Send Backward
선택된 컴포넌트의 z-index 를 1 만큼 감소시킵니다.
 
28: Send To Back
선택된 컴포넌트의 z-index 를 현재 container 의 child 중 가장 작은 값으로 감소 시킵니다.
 

Flexbox Layout

Layout type 이 flexbox 인 container 를 선택하면 다음과 같은 layout section 이 나타납니다.
flexbox layout 에 관련된 다음 3가지 CSS 속성을 이 layout section 에서 control 할 수 있습니다.
box-orient 
child 가 배치되는 방향을 수평(horizontal) 또는 수직(vertical) 으로 지정할 수 있는 속성입니다.
box-align 
box-orient 가 지정된 방향의 수직방향(Align 축 방향)에 대한 정렬을 지정합니다.
box-pack 
box-orient 로 지정된 방향으로의 정렬을 지정합니다.
 

 
1: Layout
선택된 container 의 layout type 을 flow, absolute, flexbox 중에 하나를 선택할 수 있습니다.
 
2: Orientation
box-orient 속성에 해당되며 선택된 container 의 child 가 배치되는 방향을 horizontal 또는 vertical 로 지정합니다.
 
3: Align Start
box-align 속성값을 start 로 지정합니다. Align 축의 시작 위치에 컴포넌트들이 배치됩니다.
 
4: Align Center
box-align 속성값을 center 로 지정합니다. Align 축의 중앙에 컴포넌트들이 배치됩니다.
 
5: Align End
box-align 속성값을 end 로 지정합니다. Align 축의 마지막 위치에 컴포넌트들이 배치됩니다.
 
6: Align Stretch
box-align 속성값을 stretch 로 지정합니다. 컴포넌트들의 Align 축 방향 크기가 container 만큼 늘어나게 됩니다.
 
7: Pack Start
box-pack 속성값을 start 로 지정합니다. 컴포넌트들은 box-orient 방향의 시작 위치에 배치됩니다.
 
8: Pack Center
box-pack 속성값을 center 로 지정합니다. 컴포넌트들은 box-orient 방향의 중앙에 배치됩니다.
 
9: Pack End
box-pack 속성값을 end 로 지정합니다. 컴포넌트들은 box-orient 방향의 마지막 위치에 배치됩니다.
 
10: Justify
box-pack 속성값을 jusity 로 지정합니다. 컴포넌트들은 box-orient 방향을 기준으로 일정한 간격으로 배치됩니다.
 

 Position and size for flexbox layout

 
Flexbox layout container 안에 위치하는 컴포넌트의 경우 다음과 같은 position and size section 이 나타납니다.
 

1: Flow Type
Flow type 을 block, inline-block, inline 중 하나로 선택할 수 있습니다.

2: Width
선택된 컴포넌트의 width 를 지정합니다.

3: Height
선택된 컴포넌트의 height 를 지정합니다.

4: Margin
선택된 컴포넌트의 margin 을 지정합니다.

5/6/7/8: top/right/bottom/left
선택된 컴포넌트의 top/right/bottom/left margin 을 지정합니다.

9: Padding
선택된 컴포넌트의 padding 을 지정합니다.

10/11/12/13: top/right/bottom/left
선택된 컴포넌트의 top/right/bottom/left padding 을 지정합니다.
 
14: Box Flex
선택된 컴포넌트의 box-flex 값을 지정합니다. box-flex 값이 0 보다 크면 container 의 여유공간을 box-flex 값의 비율만큼 차지하게 됩니다.

15
해당 속성 값을 px ↔ % ↔ Auto 간에 변환합니다. px ↔ % 간에 변환시에는 계산된 길이가 최대한 동일하게 변환됩니다.
 
16/17
4방향의 margin/padding 을 각각 따로 설정할 수 있는 입력 필드를 보여주거나 숨깁니다.
