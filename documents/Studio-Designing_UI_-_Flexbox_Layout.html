Flexbox layout 은 child 컴포넌트를 수평 방향 또는 수직 방향으로 배치하는 layout 방식입니다.
다양한 정렬방식을 지원하고 컴포넌트별로 flex 값을 줄 수 있기 때문에 application UI 를 만들기에 적합한 layout 방식입니다.

Panel 에 flexbox layout 적용하기

  1. 비어있는 Scene 에 Panel 을 배치합니다.
     
  2. Panel 에 Button 컴포넌트를 여러개 배치합니다.
     
  3. Panel 을 선택하고 QuickStyler > Layout 에서 Flexbox Layout 을 선택하면 다음과 같이 Button 컴포넌트가 배치됩니다.
  4. 이 상태에서 Panel 을 선택하고 QuickStyler > Layout 에서 Orientation 속성을 Horizontal 에서 Vertical 로 변경합니다.
  5. 그러면 다음과 같이 Button 이 수직으로 배치되는 것을 볼 수 있습니다.
  6.  여기서 마지막 Button 을 선택하고 palette 에서 Button 을 더블 클릭해서 Button 을 추가합니다. Visual Designer 에서 보면 다음과 같이 Button 의 크기가 다른것을 확인할 수 있습니다. 이는 flexbox layout 의 속성 (align, pack 등) 은 flow type 이 block 인 child 에만 적용되는데 방금 추가된 Button 의 layout type 은 기본적으로 inline-block 이기 때문입니다.
  7. 추가된 Button 의 flow type 을 block 으로 변경하면 align stretch 속성이 적용되어 다른 Button 과 같이 늘어나게 됩니다.

Flexbox layout panel 에서 정렬하기

 
Flexbox layout panel 에서는 orientation 방향으로의 정렬은 pack 속성이 적용되고 orientation 방향과 수직 방향으로의 정렬은 align 속성이 적용됩니다.
먼저 align 속성을 사용해서 정렬해 보도록 하겠습니다.

  1. Panel 을 선택하고 align start, align center, align end, align stretch 를 차례로 선택해 봅니다.

  2. Pack 속성을 적용해보기 전에 Panel 의 height 를 100% 로 늘려서 정렬하기 위한 여유공간을 확보합니다.
  3. Pack Start, Pack Center, Pack End, Justify 를 차례로 선택해 봅니다.


    Pack Start


    Pack Center


    Pack End


    Justify


    만일 Button 의 크기를 기본 값이 아닌 상대적인 비율로 지정하고 싶다면 BoxFlex 속성을 사용하면 됩니다.
  4. Button 중 하나를 선택하고 Position and Size 에서 BoxFlex 의 값을 1 로 변경합니다. 다음과 같이 Button 의 크기가 Panel 의 여유공간만큼 늘어나는 것을 확인할 수 있습니다.
  5. 다른 Button 을 선택한 후 BoxFlex 값을 2 로 변경합니다. 다음과 같이 Button 의 크기가 BoxFlex 가 1인 button 보다 크게 늘어나는것을 볼 수 있습니다. BoxFlex 가 0 이 아닌 값을 지정하면 여유공간을 BoxFlex 비율에 따라 나누어서 분배됩니다.
