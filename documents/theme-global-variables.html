theme 전역 변수?

Mobello에서는 Sass & Compass를 이용해서 테마를 관리하고 있습니다. 특히 Sass에서 제공하고 있는 variables는 스타일에 대해 일괄적으로 적용할 수 있기 때문에 Mobello에서는 

전역 변수로 정의해서 사용하고 있습니다.

 

전역 변수 목록

변수명타입설명
$default-container-bgcolorcolor기본 컨테이너의 배경색 설정
$default-container-colorcolor기본 컨테이너의 텍스트 색상 설정

다음과 같이 기본 컨테이너의 배경색과 텍스트 색상을 설정할 수 있습니다.

컨테이너 배경색, 텍스트색

$default-container-bgcolor: #fff;

$default-container-color: #000;

변수명타입설명
$default-colorcolor기본 텍스트 색상 설정
$selected-colorcolor선택 혹은 탭 되었을 때 색생 설정
$highlighted-colorcolor포커스 혹은 터치되고 있을 때 색상 설정
$disabled-colorcolor선택할 수 없는 상태일 때 색상 설정

상태(기본, 선택되었을 때, 터치하고 있을 때, 선택할 수 없을 때)에 대한 기본 텍스트 색상을 설정합니다.

상태별 텍스트색

$default-color: #fff;

$selected-color: #fff;

$highlighted-color: #fff;

$diabled-color: #a5a5a5;

변수명타입설명
$mask-bgcolorcolor마스크 되는 영역의 배경색
마스크 배경색

$mask-bgcolor: #e4e4e2;

변수명타입설명
$enable-gradientbooleangradient 사용 여부
$enable-border-radiusbooleanborder-radius 사용 여부
$enable-box-shadowbooleanbox-shadow 사용 여부
$default-border-radiussizeborder-radius 사용 여부가 true인 경우 설정한 크기로 기본 border-radius값을 설정한다.

$enable-gradient는 gradient를 사용할지 여부를 true | false 값으로 설정할 수 있습니다. $enable-border-radius를 false 값으로 설정하면 border가 사각형 형태로 보여집니다.

$enable-box-shadow를 true로 설정하면 shadow를 사용하지 않습니다. 모바일에서 shadow는 성능과 관련된 CSS스타일이기 때문에 지양하는 것이 좋습니다.

gradient / radius / shadow 사용여부

$enable-gradient: true;
$enable-border-radius: true;
$enable-box-shadow: true;

border-radius에 대한 기본값을 설정합니다. 

border radius 기본값

$default-border-radius: .4em;

변수명타입설명
$default-input-border-colorcolortextfield, textarea 처럼 입력하는 폼의 border 색상 설정
$default-input-highlighted-border-colorcolor포커스 혹은 터치했을 때 textfield, textarea 처럼 입력하는 폼의 border 색상 설정

extfield, textarea등 입력을 받기 위한 영역에 공통적으로 나오는 border색에 대한 설정입니다.

textfield, textarea border 색

$default-input-border-color: #959595;

$default-input-highlighted-border-color: #32BBE7;

변수명타입설명
$badge-widthsize배지 크기 설정. 높이, 너비는 동일하게 적용됨.
$badge-marginsize배지 마진 크기를 설정.
$badge-start-bgcolorcolor배지 배경색상
$badge-stop1-bgcolorcolorgradient 사용여부가 true인경우 첫번째 Stop 배경색
$badge-stop2-bgcolorcolorgradient 사용여부가 true인 경우 두번째 Stop 배경색
$badge-end-bgcolorcolorgradient 사용여부가 true인 경우 마지막 Stop 배경색

대쉬보드에서 앱 shortcut에 나오는 배지의 스타일 관련 변수들입니다.

배지 스타일

$badge-width: 24px;

$badge-margin: -($badge-width / 2) + 2px;

$badge-start-bgcolor: #F4B9BD;
$badge-stop1-bgcolor: #E7565D;
$badge-stop2-bgcolor: #E24850;
$badge-end-bgcolor: #CD4D5C;

 

cross browsing 관련한 변수들입니다. 현재 Mobello에서는 webkit 브라우저에서만 정상적으로 동작합니다. 향후 다른 브라우저에 대해서도 지원할 예정입니다.

cross browsing 관련 속성

$experimental-support-for-mozilla: false;

$experimental-support-for-opera: false;

$experimental-support-for-microsoft: false;

$experimental-support-for-khtml: false;
