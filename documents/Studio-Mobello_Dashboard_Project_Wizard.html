"Mobello Dashboard Project":/mobello-project#mobello-dashboard-project 를 생성하기 위한 wizard 입니다. Dashboard project 를 생성하면 project 에 기본적으로 포함되는 application 1개를 같이 생성하게 됩니다.

Project name
Dashboard project 의 이름을 지정합니다.

Version
Dashboard project 의 버전을 지정합니다.

Vendor
Dashboard project 의 제작자를 지정합니다.

Theme
Dashboard project 의 기본 theme 를 지정합니다.

Dashboard project 에 대한 정보를 입력하고 Next 버튼을 클릭하면 dashboard 에 포함될 application 생성을 위한 wizard page 가 나타납니다.

Application name
Application의 이름을 지정합니다. Application 이름으로는 영문자, 숫자 와 underscore(_) 만 허용됩니다.

Title
사용자에게 노출될 Application 의 제목을 지정합니다.

Version
Application 의 버전을 지정합니다.

Vendor
Application 의 제작자를 지정합니다.

Theme
Application 의 기본 Theme 를 선택합니다. 현재 mobello 에서는 default theme 와 iOS theme 를 제공하고 있습니다.
