Absolute layout 은 고정된 좌표를 기준으로 배치되는 가장 간단한 layout 방식입니다. 그러나 다양한 스크린 사이즈에 대응하기 어렵기 때문에 고정된 UI 배치가 필요한 영역에서만 한정적으로 사용하는것이 좋습니다.
 

Absolute Layout Panel

Absolute Layout 을 사용하기 위해서는 먼저 Panel 을 배치하고 layout type 을 Absolute Layout 으로 변경해주어야 합니다.

  1. 비어있는 Scene 에 Panel 을 배치합니다.
  2. QuickStyler > Position and Size 에서 height 를 50% 로 변경해서 Panel 의 크기를 늘립니다.
  3. QuickStyler > Layout 에서 Layout 속성의 값을 Absolute Layout 으로 변경합니다.

Absolute Layout Panel 에 컴포넌트 배치 

 

  1. Palette view 에서 Label 을 선택하고 Panel 에 배치합니다. 마우스를 Panel 좌측 상단에 위치시키면 다음과 같이 주황색으로 default padding 이 표시됩니다. 마우스를 클릭하면 Label 이 padding 위치에 배치됩니다.
     
     
     
  2. 마찬가지로 TextField 컴포넌트를 Label 옆에 배치합니다. Label 오른쪽에 마우스를 올리면 다음과 같이 default margin 이 적용되어 컴포넌트간에 일정한 간격을 유지할 수 있습니다.

  2. QuickStyler 의 Position and Size section 에서 Align Right 를 선택하면 다음과 같이 오른쪽 정렬 됩니다.




    마찬가지로 Position and Size 에서 Align Left, Align Center 등을 선택하면 선택된 컴포넌트들을 좌측 정렬 하거나 중앙 정렬 할 수 있습니다.

Width and Height

 

  1. Label 을 선택한 후 Designer 에서 마우스 드래그로 크기를 조정하면 Position and Size 에 변경된 크기값이 반영되는 것을 확인할 수 있습니다.

  2. 여기서 width 속성의 px 버튼을 누르면 현재 크기에 대응되는 % 값으로 변환됩니다.



    컴포넌트 크기가 % 로 지정되면 상위 Panel 에 대한 상대크기로 계산되기 때문에 상위 Panel 의 크기가 변경되면 해당 컴포넌트의 크기도 같이 변경되게 됩니다.
  3. 여기서 % 버튼을 한번더 누르면 값이 auto 로 변경 됩니다.



    Width,height 값이 auto 이면 컴포넌트의 content 에 따라 크기가 자동으로 조정되게 됩니다. 예를 들어 Label 컴포넌트의 width 로 auto 를 지정하면 text 값에 따라 크기가 달라지게 됩니다.

    Ctrl 키로 여러개의 컴포넌트를 선택하고 Position and Size 의 Match Width 버튼을 누르면 가장 마지막에 선택된 컴포넌트를 기준으로 width 값이 동일하게 변경됩니다.



    마찬가지로 Match Height, Match Size 로 height 또는 width,height 모두를 동일하게 맞출 수 있습니다.

좌표 기준 변경


Absolute Layout 에서는 기본적으로 top, left 를 기준으로 컴포넌트의 위치가 결정되며 container 의 크기가 변하면 컴포넌트와 container 오른쪽 하단 (right bottom) 과의 간격이 달라지게 됩니다.
만일 container 의 크기와 상관없이 container 우하단과 컴포넌트와의 거리를 일정하게 유지하려면 bottom, right 의 값을 px 로 변경해야 합니다.

다음은 top,left 를 px 값으로 지정하고 botttom,right 를 auto 로 지정한 경우를 나타내는 그림입니다.


다음은 top,left 를 auto 로 지정하고 bottom,right 를 px 값으로 지정한 경우를 나타내는 그림입니다.


만일 container panel 의 크기가 늘어날때 component 의 크기도 함께 늘어나도록 하려면 top,left,right,bottom 을 px 값으로 지정하고 width,height 를 auto 로 지정하면 됩니다.

다음은 top,left,right,bottom 을 px 값으로 지정하고 width,height 를 auto 로 지정한 경우를 나타내는 그림입니다.

좌표 기준을 bottom,right 로 변경
  1. Button 을 Panel 오른쪽 하단에 배치합니다.
  2. QuickStyler 의 Position and Size 에서 bottom 에 해당되는 부분과 right 에 해당하는 부분을 모두 클릭합니다.
  3. 다음과 같이 left,top 가 auto 로 변경되고 right,bottom 은 px 값으로 변경된 것을 확인할 수 있습니다.
  4. Toolbar 에서 viewport size 를 변경해보면 화면 크기가 늘어남에 따라 Button 컴포넌트의 위치도 변경되는 것을 볼 수 있습니다.
