Mobello Studio 에서는 다음과 같이 Visual Designer 와 그에 관련된 View 들을 사용해서 화면을 쉽게 만들 수 있도록 지원하고 있습니다.

  • Visual Designer
    Design perspective 의 주요 editor 로 palette 를 이용하여 mobello project 디자인을 할 수 있습니다.
  • Outline view 
    선택 된 designer 에 포함된 component 들의 구조적인 개요를 나타냅니다. 컴포넌트들의 배치 순서와 포함 관계를 파악할 수 있습니다.
  • Palette view 
    디자인 가능한 component 리스트를 나타내는 View. 이 Palette view 를 이용해 D&D 방식으로 손쉽게 디자인 할 수 있습니다.
  • QuickStyler View 
    빠른 스타일 입력을 위한 View. 폰트 속성이나 색상, 위치 관련 정보를 수정할 수 있습니다.
  • Attributes View
    Component 상세 정보 변경을 위한 View. Component 자체 속성 및 CSS, Event 정보를 수정할 수 있습니다.
