Attributes view 에서는 컴포넌트의 모든 속성을 편집할 수 있습니다.
 

 
1: Show Categories
속성들을 category 로 분류해서 보여줍니다.
 
2: Summarized
값이 설정된 속성들만 보여주고 값이 없는 속성들은 감춥니다.
 
3: Show Advanced Properties
감추어진 고급 속성들을 추가적으로 보여줍니다.
 
4: Remove
속성을 삭제합니다. 속성을 선택하고 Delete 키를 사용해도 됩니다.
 
5: Expand All
Category 하위 속성들을 모두 펼쳐 보여줍니다.
 
6: Collapse All
Category 를 모두 접습니다.
 
7: type filter text
입력한 단어를 포함하는 속성들만 보여줍니다. Esc 키로 취소할 수 있습니다.
 
8: General
일반적인 속성들을 보여줍니다.
 
9: CSS
CSS 스타일 속성들을 보여줍니다.
 
10: Events
Event 속성들을 보여줍니다.
 
11: Computed Style
선택된 DOM 요소의 계산된 스타일(computed style) 만 보여줍니다.
 
12: Attributes Viewer
속성을 선택하고 값을 편집할 수 있는 영역입니다. 속성들은 category 로 분류해서 보여지게 되며 category 항목을 더블 클릭해서 접거나 펼칠 수 있습니다.
 
13: Description Area
선택된 속성에 대한 설명과 기타 정보들을 보여줍니다.
