New Wizard

 
New Wizard 를 통해 새로운 project 나 file을 추가할 수 있습니다. File → New 메뉴 또는 Toolbar 의 new 버튼 () 을 사용해서 wizard 를 선택하고 호출할 수 있습니다.
 
Mobello Studio 에서는 다음과 같은 New Wizard 들을 제공합니다.
 

 

  • Mobello Application Project Wizard
  • Application Template Page
  • Mobello Application Wizard
  • Mobello Dashboard Project Wizard
  • Mobello Scene File Wizard
  • JavaScript Source File Wizard
     

Project Explorer View

Project explorer view 에서는 project 와 file 을 보여주고 컨텍스트 메뉴를 통해서 삭제, 복사, 이름 변경 등 여러가지 기능을 제공합니다.

1: Collapse All
Project explorer view 에 펼쳐져 있는 folder 를 모두 접습니다.

2: Link with Editor
현재 열려 있는 editor 의 file 과 project explorer view 에 선택된 file을 동기화 시켜줍니다.
