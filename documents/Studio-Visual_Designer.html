Visual Designer 는 scene file 을 디자인할 수 있는 editor 로서 Mobello Studio 에서 가장 중요한 UI 입니다.
 

Toolbar

 
Visual designer 를 열었을 때 활성화 되는 toolbar 버튼을 설명합니다.
 

 
1: New ()
새로운 project 와 file 을 생성하는 new wizard 를 띄웁니다.
 
2: Save ()
파일을 저장합니다.
 
3: Save All ()
열려있는 모든 파일을 저장합니다.
 
4: Debug ()
현재 선택된 project 를 debug 모드로 실행합니다.
 
5: Run ()
현재 선택된 project 를 실행합니다.
 
6: Change Editor Split Orientation ()
Visual Designer 에 Design/source page 가 존재하는 경우 split bar 의 방향과 위치를 변경합니다.
Visual Designer 에 Design/source page 를 추가하기 위해서는 Mobello Studio Preferences Page > Visual Designer > Scene Designer 에서 Show source/design page 를 선택합니다.
 
7: Open with Web Browser ()
현재 선택된 HTML 을 웹 브라우저에서 엽니다.
 
8: Zoom ()
콤보를 사용해서 visual designer 의 확대/축소 배율을 조정할 수 있습니다.
 
9: Zoom In ()
Visual designer 를 확대합니다.
 
10: Zoom Out ()
Visual designer 를 축소합니다.
 
11: Viewport size

Visual designer 의 viewport size 크기를 변경합니다. Viewport size 항목을 추가/삭제 하기 위해서는 마지막 Configure viewport sizes.. 를 선택합니다.
참고: "Mobello Studio Preferences Page > Visual Designer > Viewport Size":http://mobello.github.com/studio-preferences-ko/#viewport_size
 
12: Undo ()
방금 실행한 명령을 취소합니다.
 
13: Redo ()
Undo 로 취소한 명령을 다시 수행합니다.
 
14: Show Visual Aid for Empty Container ()
이 버튼이 선택된 경우 실제 브라우저에서는 보이지 않는 요소라고 하더라도 디자인 작업 편의를 위해 크기를 가지며 가상의 border 가 표시됩니다.
 
15: Refresh ()
Visual designer 의 source page 에서 변경한 사항이 design page 에 제대로 반영이 안되는 경우 새로 고침을 합니다.
 
 

Context Menu

 
Visual Designer 에서 마우스 오른쪽 버튼을 클릭하면 다음과 같이 context menu 가 나타납니다.
 

 
Undo
방금 실행한 명령을 취소합니다.
 
Redo
방금 Undo 로 취소한 명령을 다시 실행합니다.
 
Copy
현재 선택된 요소를 clipboard 로 복사합니다.
 
Paste
Clipboard 로 복사한 내용을 현재 마우스 위치에 붙여넣습니다.
 
Delete
현재 선택된 요소를 삭제합니다.
 
Select Container
현재 선택된 요소의 부모를 선택합니다.
 
Select Next Sibling
현재 선택된 요소의 다음 요소를 현재 선택 영역에 추가합니다.
 
Select Previous Sibling
현재 선택된 요소의 이전 요소를 현재 선택 영역에 추가합니다.
 
Select All Sibling
현재 선택된 요소 부모의 모든 자식 요소를 선택합니다.

Alt+Shift+Left

현재 선택된 요소 부모의 모든 자식들을 선택

Alt+Shift+Up
