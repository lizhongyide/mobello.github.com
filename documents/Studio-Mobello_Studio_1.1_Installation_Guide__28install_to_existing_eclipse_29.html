This procedure installs the Mobello Studio to existing eclipse installation.

Prerequisite

  • Mobello Studio 1.1 requires eclipse 3.7 (Indigo)

Procedure

  1. Start eclipse, then select Help > Install New Software... in the dialog that appears, enter the update site URL into the Work with text box:

     http://mobello.github.com/mobello-studio/updates

    And press enter key.
     
  2. You should see the Mobello Studio. Select the checkbox and click Next.
     
     

  3. Review the features that you are about to install. Click Next.

     
     
  4. Read the license agreements and then select I accept the terms of the license agreements. Click Finish.


     
  5. If you encounter the following warning, then press OK to proceed.



  6. You will then be asked if you would like to restart Eclipse. Click Restart now.


     
  7. Now you are ready to build Mobello application!
