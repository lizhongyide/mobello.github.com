Class tau.rt.EventPublisher

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventPublisher
  • Defined in:
    tau.rt.js

EventPublisher는 이벤트가 발생되면 해당 이벤트 리스너와 바인딩 시키는 작업을 수행한다.

이 클래스의 주요 기능은 callback function 등록 및 관리하고 이벤트가 발생될 시점에 해당 callback function들이 정상적으로 호출될 수 있도록 한다.

또한 이벤트 발생과정의 상태, 기본 동작, 예외상황처리 등을 처리한다. 예를 들어, 사용자가 이벤트를 더이상 propagation시키지 않고자 할 경우 stopPropagation()메소드를 호출하여 더이상 이벤트가 bubbling되지 않도록 할 수 있다.
