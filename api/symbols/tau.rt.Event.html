Class Detail

tau.rt.Event(name, source, touches, targetTouches, changedTouches)
Mobello에서는 자체정의된 이벤트 시스템을 사용한다. Mobello에서 제공하는 UI 컴포넌트로 부터 발생되는 다양한 이벤트는 tau.rt.Event 객체를 나타낸다.

UI관련 대부분의 클래스는 tau.rt.EventDelegator를 상속받아 정의된다.

Parameters:
{String} name
이벤트 이름
{tau.rt.EventDelegator} source
이벤트가 발생되는 tau.rt.EventDelegator 객체
{Array} touches
현재 화면을 터치했을 때 손가락별 터치정보
{Array} targetTouches
동일한 노드에서 시작한 터치정보를 필터일한다.
{Array} changedTouches
이벤트를 최초 발생시킨 터치정보

Method Detail

  • alwaysBubble()
    발생된 이벤트가 적어도 한개의 이벤트 리스너에 의해 처리되었다면 이 이벤트는 더이상 propagation되지 않는다. 하지만 alwaysBubble() 메소드를 호출하면 이벤트가 처리되더라도 계속해서 Bubbling되도록 한다.
  • {String} getName()
    현재 이벤트 이름을 반환한다.
    Returns:
    {String} 이벤트 명

  • {Object} getSource()
    이벤트가 발생된 원본객체(tau.rt.EventDelegator)를 반환한다.
    Returns:
    {Object} EventDelegator 객체

  • preventDefault()
    이벤트의 default function이 수행되지 않도록 한다.
  • stopPropagation()
    이후의 이벤트 propagation작업을 취소한다. 따라서 Bubbling up단계에 등록된 이벤트 리스너는 동작하지 않는다.
