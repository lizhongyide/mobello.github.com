Class tau.ui.Select

HTML의 select tag를 확장한 컴포넌트.

현재 선택된 값이 화면에 출력되고 값을 변경하고자 할 때 Select 영역을 탭하면 팝업으로 리스트가 나와 다중값을 선택하거나, 하나의 값만 선택한다.

대표적인 이벤트 :

#event:tau.rt.Event.VALUECHANGE
옵션을 선택하고 팝업을 닫으면 이벤트를 발생시킨다.

Method Detail

  • {Boolean} add(option, index, immediate)
    컴포넌트를 하위 아이템으로 추가한다.

    컴포넌트의 부모/버블링 관계 설정을 하고 컴포넌트에서 이벤트를 처리하지 않아도 현재 인스턴스에서 처리하도록 한다. immediate가 false인 경우 tau.rt.Event.COMPADDED 이벤트를 발생시킨다.

    Parameters:
    {tau.ui.Button | Object} option
    현재 인스턴스에 추가할 버튼 컴포넌트 객체 혹은 버튼 컴포넌트 생성자 옵션. Object인 경우 옵션으로 tau.ui.Button을 생성한 후 추가한다.
    {Number} index Optional
    특정 위치에 컴포넌트 DOM element를 추가한다.
    {Boolean} immediate Optional, Default: false
    컴포넌트를 바로 draw할지 설정한다. false이면 객체가 draw할 때 일괄적으로 DOM Tree에 반영된다.
    Throws:
    {TypeError}
    명시된 comp가 tau.ui.Button의 객체가 아닐 경우
