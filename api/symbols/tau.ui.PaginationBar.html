Class tau.ui.PaginationBar

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Component
         |
         +--tau.ui.PaginationBar
  • All Mixin Classes:
    tau.ui.Container
  • Defined in:
    tau.ui.comp.js

PaginationBar는 tau.ui.Table에서 페이지네이션 처리를 위해 사용되는 클래스이다.

제공하는 페이지네이션 타입:

  • tau.ui.PaginationBar.NORMAL_TYPE: 이동할 페이지 번호이 버튼으로 제공
  • tau.ui.PaginationBar.SLIDER_TYPE: slider를 움직여서 해당 페이지로 이동 제공

대표적인 기능 :

  • #setFirstLastButtonVisible: 맨처음, 맨마지막 페이지로 가는 버튼 제공
  • #setPrevNextButtonVisible: 이전, 다음페이지로 가는 버튼 제공
