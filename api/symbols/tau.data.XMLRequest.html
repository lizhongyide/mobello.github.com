Class Detail

tau.data.XMLRequest(opts)
명시된 옵션들을 이용하여 새로운 객체를 생성한다. Request 처리에 따라 다음과 같은 시스템 이벤트가 발생된다.

tau.rt.Event.REQ_SENT: Request가 전송되었을 때
tau.rt.Event.REQ_RECEIVED: Response를 받았을 때
tau.rt.Event.REQ_ABORTED: Request가 중지되었을 때

시스템 이벤트를 받기 위해서는 다음과 같이 Runtime에 이벤트 리스너를 달아야 한다. 

var rt = tau.getRuntime();
rt.onEvent(tau.rt.Event.REQ_SENT, function (e, payload) {
  ...
});
Parameters:
{Object} opts Optional
XMLHttpRequest 요청에 필요한 옵션들
{String} opts.type Optional, Default: 'GET'
Request 방식(GET, POST중 택일)
{String} opts.contentType Optional, Default: 'application/x-www-form-urlencoded'
콘텐츠 인코딩 방식
{String} opts.encoding Optional, Default: 'UTF-8'
인코딩 방식
{Boolean} opts.async Optional, Default: true

{Boolean} opts.silent Optional, Default: false

{String} opts.url Optional
Request를 전송하기 위한 URL
