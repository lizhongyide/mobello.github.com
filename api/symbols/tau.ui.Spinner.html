Class tau.ui.Spinner

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Component
         |
         +--tau.ui.Spinner
  • Defined in:
    tau.ui.comp.js

Spinner 클래스는 값을 설정된 범위에서 증가 혹은 감소시키는 컴포넌트이다.

Spinner에서는 증가버튼, 감소버튼을 제공해서 버튼을 탭하거나, 터치를 계속하면 값이 계속 증가하거나, 반대로 계속 감소한다. 또한 값을 직접 입력해서 값을 바로 설정도 가능하다.

대표적인 기능 :

  • 값을 설정(#setValue)하고, 값을 가져오는 기능(#getValue, #getRawValue)
  • tau.ui.Spinner.acceleration : 증가 혹은 감소 버튼을 탭하고 계속 누르고 있으면 가속하는 기능

대표적인 이벤트 :

event:tau.rt.Event.VALUECHANGE
증가, 감소 버튼을 탭해서 값이 변경하거나 직접 입력 필드에서 값을 변경했을 때 발생한다.
