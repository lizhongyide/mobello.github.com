Class tau.ui.IndexBar

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Component
         |
         +--tau.ui.IndexBar
  • All Mixin Classes:
    tau.ui.Control
  • Defined in:
    tau.ui.comp.js

인덱스바는 테이블의 섹션으로 빠르게 이동하는데 사용된다.

사용자가 인덱스바의 인덱스 문자를 탭하거나 인덱스바를 터치했다가 이동하면서 특정 인덱스 문자에서 터치를 종료하면 인덱스 문자를 가지고 있는 tau.ui.TableSection으로 스크롤 이동한다.

현재 tau.ui.Table에서만 사용되고 있다.
