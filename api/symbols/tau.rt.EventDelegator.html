Class tau.rt.EventDelegator

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
  • Direct Known Subclasses:
    tau.rt.Module, tau.rt.Runtime, tau.ui.Component, tau.ui.Controller, tau.ui.Scene, tau.ui.SystemDialog
  • Defined in:
    tau.rt.js

EventDelegator는 이벤트를 전송하고 받기위해 필요한 허브역할을 수행한다. 뿐만 아니라 상위 EventDelegator와 Bubbling관계를 형성한다.

이 클래스는 다른 Mobello 클래스 정의시 상속받을 수 있으며 이를 통해 이벤트를 publish하고, fire할 수 있으며 또한 등록된 이벤트에 대해 listening을 수행할 수 있다.

  • {Boolean} fireEvent(e, payload, delay)
    명시된 이벤트 이름 또는 이벤트 객체를 이용하여 이벤트를 발생시킨다.

    이벤트가 발생되면 capture단계와 bubbling-up단계를 거치면서 등록된 리스너들을 실행시킨다.

    파라미터 중 delay가 0이상의 값을 가질 경우 Async방식으로 등록된 이벤트 리스너가 호출되지만 -1일 경우 Sync 방식으로 이벤트 리스너가 실행되어 시간지체가 발생되지 않는다.

    Parameters:
    {String|tau.rt.Event} e
    이벤트명 또는 이벤트 객체
    {Object} payload
    이벤트 발생시 같이 전송될 데이터 객체
    {Number} delay
    실제 이벤트가 발생될 때까지의 시간(milliseconds)
    Throws:
    {TypeError}
    이벤트의 source객체가 현재 인스턴스와 다를 경우

    Returns:
    {Boolean} 정상적으로 이벤트가 발생되면 true를 반환
