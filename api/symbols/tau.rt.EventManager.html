tau.rt.EventManager(opt, touchMoveThreshold, root)
명시된 파라미터를 이용하여 새로운 객체를 생성한다. EventManager객체는 tau.rt.Runtime에서 Signleton으로 생성, 관리된다.
Parameters:
{Object} opt
Event Manager 생성 옵션
{Boolean} opt.enabled Optional, Default: true
false이면 Mobello의 이벤트 시스템이 동작하지 않음
{Number} touchMoveThreshold Optional, Default: 5
Touch Move이벤트가 발생할 때 까지의 최소시간
{Object} root
이벤트를 리스닝하기 위한 Root DOM 앨리먼트
