Class tau.ui.DashboardController

  • HIERARCHY

Dashboard의 UI를 구성하는 SceneController 클래스이다. DashboardController 클래스는 Dashboard를 구현하기 위해 UI Framework에서 기본으로 사용하는 SceneController이며, 사용자가 직접 UI를 커스터마이징을 하기 위해서는 tau.ui.DashboardController를 상속받아 구현해야 한다.

  ...
});

{
  ...
  dashboard: {controller: 'foo.ExtendedController'}
  ...
}
