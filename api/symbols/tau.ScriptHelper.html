Class tau.ScriptHelper

  • HIERARCHY
    tau.TObject
   |
   +--tau.ScriptHelper
  • Defined in:
    tau.core.js

동적인 방법으로 자바스크립트(opts.type = 'script') 또는 CSS 링크(opts.type = 'css')를 로딩한다. 동적으로 자바스크립트를 로딩하기 위해 document의 head 앨리먼트 내에 script 또는 lilnk 앨리먼트를 생성하는 형태로 해당 스크립트를 로딩한다.

Class Detail

tau.ScriptHelper(opts)
명시된 옵션들을 이용하여 외부의 자바스크립트 또는 CSS파일을 동적으로 로딩한다. 
<script type="text/javascript" src="..."></script>
<link type="text/css" rel='stylesheet' media="screen" href="..."></link>
Parameters:
{Object} opts
ScriptHelper 객체 생성시 사용할 수 있는 옵션들
{String} otps.type Optional, Default: 'script'
'css' 또는 'script'
{String} otps.url Optional
스크립트에 대한 URL 주소, 만약 생략하면 load() 메소드를 통해 기술할 수 있다.
{Function} otps.callbackFn Optional
성공적으로 로딩되었을 때 호출되는 Callback 메소드
{Function} otps.timeoutFn Optional
타임아웃이 되었을 때 호출되는 Callback 메소드
{Number} otps.timeout Optional
스크립트가 로딩 완료될때 까지의 시간. millisecond로 표현한다.
{Object} otps.context Optional
callback 함수 내에서 사용될 'this' 객체
{Boolean} otps.noCache Optional, Default: false
기존에 동일한 스크립트 파일이 로딩되어 있더라도 새롭게 로딩한다.
{Boolean} otps.autoUnload Optional, Default: false
로딩이 완료되면 로딩된 스크립트를 자동으로 unloading한다.
{String} otps.charset Optional
Characterset 인코딩을 지정한다. 생략하면 setCharset()메소드를 통해 지정할 수 있다.

  • {Element} load(url, callbackFn, timeoutFn)
    head 앨리먼트 하부에 script앨리먼트를 동적으로 삽입함으로써 해당 스크립트를 동적으로 로딩한다. 해당 스크립트가 성공적으로 로딩되었을 때 callbackFn 메소드가 호출된다.

    Parameters:
    {String} url
    로딩할 JavaScript 소스 URL
    {Function} callbackFn Optional
    로딩이 완료된 후에 호출될 callback 메소드
    {Function} timeoutFn Optional
    Timeout이 발생되었을 경우 호출될 callback 메소드
    Returns:
    {Element} 새롭게 추가된 DOM 앨리먼트
