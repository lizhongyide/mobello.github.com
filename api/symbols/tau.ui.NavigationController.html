Class tau.ui.NavigationController

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Controller
         |
         +--tau.ui.NavigationController
  • Defined in:
    tau.ui.ctrl.js

Navigation관련 기능들을 정의한다. Scene Navigation관련 클래스를 작성하기 위해서는 NavigationController 클래스를 상속받아 구현한다. 일반적으로 많이 사용되는 Scene Navigation관련 클래스는 tau.ui.SequenceNavigator와 tau.ui.ParallelNavigator가 있다.
