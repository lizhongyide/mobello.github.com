Class tau.ui.Slider

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.EventDelegator
      |
      +--tau.ui.Component
         |
         +--tau.ui.Slider
  • All Mixin Classes:
    tau.ui.Control
  • Direct Known Subclasses:
    tau.ui.Switch
  • Defined in:
    tau.ui.comp.js

Slider는 값의 지속적인 범위에서 하나의 값을 선택하는 데 사용되는 컴포넌트이다.

Slider 바 축을 따라서 thumb을 이동 혹은 바를 터치해서 값을 선택할 수 있다.

대표적인 기능 :

  • tau.ui.Slider.vertical: 슬라이더 표시 방향 설정 기능
  • #setTickSize: 슬라이더 thumb를 터치해서 이동했을 때 틱사이즈 만큼 이동하는 기능
  • tau.ui.Slider.tappable: 슬라이더 바를 탭하면 틱사이즈만큼 이동하는 기능

대표적인 이벤트 :

#event:tau.rt.Event.VALUECHANGE
Slider의 thumb을 이동시키거나, #setValue 메소드를 통해 값이 변경되었을 때 발생하는 이벤트이다.

tau.ui.Slider()
생성자, 새로운 Slider객체를 생성한다. 객체 생성시 슬라이더 표시방향, 최소값, 최대값, 초기화값, 틱 사이즈를 이용하여 객체를 생성한다. 
// slider 생성및 이벤트 등록
loadScene: function () {
  var scene = this.getScene();
  // 수직방향의 Slider 컴포넌트를 생성한다. (최소값 0, 최대값 200, 초기값 50)
  var hSlider = new tau.ui.Slider({
    id: 'hSlider',
    minValue: 0,
    maxValue: 100,
    tickSize: 10,
    threshold: true
  });
  
  // 수직방향의 Slider 컴포넌트를 생성한다. (최소값 0, 최대값 200, 초기값 50) 
  var vSlider = new tau.ui.Slider({
    id: 'hSlider',
    vertical: true, 
    minValue: 0, 
    maxValue: 200, 
    value: 50
  };

  // valueChanged 이벤트를 처리한다.
  vSlider.onEvent(tau.rt.Event.VALUECHANGE, this.handleValueChange, this);
  hSlider.onEvent(tau.rt.Event.VALUECHANGE, this.handleValueChange, this);
  
  scene.setComponents([hSlider, vSlider]);
},
// 값이 변경되었을 때 처리하는 부분
handleValueChange: function (e, payload) {
  var src = e.getSource();
  tau.log.info(src.getId());    // slider id
  tau.log.info(payload.oldValue); // 변경 이전값 
  tau.log.info(payload.newValue); // 현재값
}
