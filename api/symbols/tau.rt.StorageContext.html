Class Detail

tau.rt.StorageContext(name, reloadOnRead, platformId)
명시된 파라미터를 이용하여 새로운 객체를 생성한다. 인덱스로 사용될 name 값을 이용하여 localStorage에 데이터를 저장하고 추출한다.

localStorage는 객체 저장을 지원하지 않지만 내부적으로 저장된 객체를 stringify메소드를 통해 문자열로 변환한 다음 localStorage객체에 저장한다.

Parameters:
{String} name
Storage context 이름
{Boolean} reloadOnRead
true이면 값을 가져올 때 마다 localStorage를 refresh한다.
{String} platformId
유일한 아이디 값
Throws:
{Error}
브라우저가 localStorage를 지원하지 않을 경우
