Class Detail

tau.URLHelper(context, params)
URL을 편리하게 조작할 수 있도록 하는 Helper 클래스이다. 이 클래스를 좀더 편리하게 사용할 수 있도록 tau.url() Utility 메소드를 지원한다. 
var url = new tau.URLHelper('http://foo.com');
url.protocol(); // 'http:'
url.host(); // 'foo.com'
Parameters:
{String} context
URL 문자열
{Object|String} params Optional
URL에 추가할 파라미터
See:
tau.url

Method Detail

  • {Object} appendParam(name, value, overwrite)
    URL의 파라미터를 추가하거나 기존의 것을 덮어쓴다. 
    var url = new tau.URLHelper('foo.html', 'aaa=bbb');
url.appendParam('ccc', 'ddd');
url.query(); // "aaa=bbb&ccc=ddd"
    Parameters:
    {String} name
    파라미터 이름
    {String} value
    파라미터 값
    {Boolean} overwrite
    true이면 기존의 값을 덮어쓴다.
    Returns:
    {Object} 'this' 객체

  • {String} host()
    URL에 명시된 호스트 이름을 반환한다. 
    var url = new tau.URLHelper('http://foo.com');
url.host(); // 'foo.com'
    Returns:
    {String} 호스트 명

  • {Boolean} isRemote()
    URL이 현재 Document를 기준으로 원격 도메인인지 확인하고 그 결과를 true/false로 반환한다.
    Returns:
    {Boolean} 만약 URL이 원격도메인이면 true를 반환한다.

  • {String|Array} param(name, all)
    URL에서 명시된 파라미터 키에 해당하는 값들을 모두 반환한다. 
    var url = new tau.URLHelper('http://foo.com', 'aaa=bbb&ccc=ddd&ccc=eee');
url.query(); // "aaa=bbb&ccc=ddd&ccc=eee"
url.param('ccc'); // "ddd"
url.param('ccc', true); // ["ddd", "eee"]
    Parameters:
    {String} name
    찾고자 하는 파라미터 이름
    {Boolean} all
    true이면 일치하는 모든 파라미터의 값을 배열로 반환한다.
    Returns:
    {String|Array} 하나 또는 배열형태의 파라미터 값

  • {Object} paramMap(all)
    URL에 명시된 파라미터의 키와 값을 맵 형태로 반환한다. 
    var url = new tau.URLHelper('http://foo.com', 'aaa=bbb&ccc=ddd&ccc=eee');
url.paramMap(); // {"aaa":"bbb","ccc":"ddd"}
url.paramMap(true) // {"aaa":["bbb"],"ccc":["ddd","eee"]}
    Parameters:
    {Boolean} all
    true이면 해당키와 매핑되는 모든 값을 반환한다.
    Returns:
    {Object} 파라미터 맵

  • {String} protocol()
    URL에 사용된 프로토콜을 반환한다. 
    var url = new tau.URLHelper('http://foo.com');
url.protocol(); // 'http:'
    Returns:
    {String} 프로토콜 이름

  • {String} query()
    URL에 사용된 Query String을 반환한다. 
    var url = new tau.URLHelper('http://foo.com', {aaa: 'bbb', ccc: 'ddd'});
url.query(); // "aaa=bbb&ccc=ddd"
    Returns:
    {String} Query string

  • {Object} resolve(base)
    상대경로의 URL을 절대경로의 URL로 변환한다.

    만약 base가 설정되어 있고 URL이 상대경로이면 리턴되는 결과는 base가 prepend된 형태가 된다.

    상대경로는 http, ftp 등과 같은 prefix로 기술되지 않고 ./, ../, /, //와 같이 기술된다. 

    var url = new tau.URLHelper('foo.html', 'aaa=bbb&ccc=ddd&ccc=eee');
url.resolve('http://aaa.bbb/');
// prints "http://aaa.bbb/foo.html?aaa=bbb&ccc=ddd&ccc=eee"
    Parameters:
    {String} base
    prepend할 base URL, 만약 null이면 '/'을 사용한다.
    Returns:
    {Object} 'this' 객체
