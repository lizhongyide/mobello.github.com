Class tau.rt.Application

개발자들이 개발한 앱을 추상화한 클래스이며 관련된 기능들을 정의한다.

Application은 관련된 모듈을 Async방식으로 로딩한다. 따라서 Application객체의 start메소드가 호출되어다 하더라도 즉시 실행되는 것이 아니라 Application에서 필요로 하는 리소스가 모두 로딩된 후에 Lifecycle을 시작한다.
