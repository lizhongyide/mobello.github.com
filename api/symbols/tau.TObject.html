Class Detail

tau.TObject()
$class()를 이용하여 생성하는 모든 클래스는 내부적으로 tau.TObject를 상속받는다. 따라서 다음과 같이 instanceof 를 통해 Mobello 객체인지 확인할 수 있다. 
var foo = // Mobello 클래스 인스턴스 
if (foo instanceof tau.TObject) {} // true

  • {Object} currentStack()
    현재 callstack의 정보를 반환한다. 이 메소드의 호출이 빈번히 발생할 경우 성능저하가 발생할 수 있으므로 신중하게 사용해야 한다. 
    var aa = ...

    Returns:
    {Object} 현재의 callstack 정보를 담고있는 객체

  • {Boolean} isMixinOf(target)
    명시된 클래스가 현재 객체에 mixin되었는지 여부를 확인하고 결과를 true/false로 반환한다. 
    $class('foo.Bar').mixin(foo.Baz).define({
  ...
});
    var bar = new foo.Bar();
bar.isMixinOf(foo.Baz); // true
    Parameters:
    {Object} target
    현재 객체에 mixin이 적용되었는지 확인하고자 하는 클래스
    Returns:
    {Boolean} 현재 객체에 명시된 클래스가 mixin되었다면 true를 그렇지 않으면 false를 반환한다.

  • {String} toString()
    현재 객체를 문자열로 변환하여 반환한다. 일반적으로 Mobello클래스가 foo.Bar로 정의되었다면 반환되는 문자열은 "[object foo.Bar]"가 된다. 
    var v = new foo.Bar();
v.toString(); // '[object foo.Bar]'
    Returns:
    {String} 객체를 표현하는 문자열
