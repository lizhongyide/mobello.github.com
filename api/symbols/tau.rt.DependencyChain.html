Class tau.rt.DependencyChain

  • HIERARCHY
    tau.TObject
   |
   +--tau.rt.DependencyChain
  • Defined in:
    tau.rt.js

연관성이 있는 클래스들을 로딩하기 위한 Chain을 구성하고 resolve 메소드를 호출하면 Chain으로 구성된 라이브러리들을 순서대로 로딩될 수 있도록 보장한다.
